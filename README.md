## 🌟 Overview

This project is a **real-world AI solution** designed to help small businesses automate customer interactions.

It simulates a WhatsApp chat system where users can:
- Ask questions  
- Get instant replies  
- Receive product recommendations  

⚡ Powered by modern AI (NLP), this assistant behaves like a **human sales agent—available 24/7**.

---

## 🚨 Problem

Small businesses face major communication challenges:

- ⏳ Slow response to customers  
- 📉 Lost sales opportunities  
- 💬 Too many chats to manage  
- 🌙 No 24/7 availability  

---

## 💡 Solution

This project introduces an **AI-driven assistant** that:

- 🤖 Automatically replies to messages  
- 🧠 Understands customer intent  
- 🛍️ Recommends products  
- ⚡ Responds instantly
